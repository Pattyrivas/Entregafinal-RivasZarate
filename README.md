# 🍕 Simulador de Pedidos — Pizzería Ficticia

Aplicación web que simula el proceso de pedido en una pizzería: selección de productos, armado de carrito y cálculo de totales en tiempo real. Proyecto final del curso de JavaScript en Coderhouse.

🔗 **Demo en vivo:** https://pattyrivas.github.io/Entregafinal-RivasZarate/

## ✨ Funcionalidades

- Catálogo de pizzas y productos cargado desde JSON
- Carrito de compra con suma de subtotales y total
- Manipulación dinámica del DOM
- Validaciones de interacción del usuario
- Diseño responsive con CSS

## 🛠️ Stack Tecnológico

- **JavaScript (Vanilla ES6+)**
- **HTML5**
- **CSS3**
- **JSON** — fuente de datos del catálogo

## 🚀 Cómo ejecutar el proyecto localmente

```bash
# Clonar el repositorio
git clone https://github.com/Pattyrivas/Entregafinal-RivasZarate.git

# Entrar a la carpeta
cd Entregafinal-RivasZarate

# Abrir index.html en el navegador
# (o usar la extensión Live Server de VS Code)
```

No requiere instalación de dependencias: es JavaScript puro.

## 📌 Aprendizajes aplicados

Proyecto enfocado en lógica de programación: manipulación del DOM sin frameworks, manejo de eventos, estructuras de datos (arrays/objetos) y cálculo dinámico de totales — base sólida antes de avanzar a React.
